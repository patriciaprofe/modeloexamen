Modelo de Parcial

Desarrollo de Landing Page Responsive con HTML y CSS

Consigna

En el siguiente enlace encontrarán un documento PDF que contiene el wireframe de una landing page. A partir de ese diseño deberán desarrollar una página web funcional utilizando únicamente HTML y CSS.

Requisitos de Desarrollo

  • Todo el código debe estar contenido en un único archivo llamado index.html.
  • El documento deberá incluir tanto la estructura HTML como los estilos CSS dentro del mismo archivo.
  • No se permite utilizar frameworks, librerías ni plantillas prediseñadas.
  • El diseño final debe respetar la estructura y organización visual planteada en el wireframe.
  • La paleta de colores, tipografías y estilos visuales podrán ser definidos por cada estudiante.
  • Las imágenes deberán ser invocadas desde los enlaces proporcionados en la consigna.
  • El sitio debe ser completamente responsive, adaptándose correctamente a dispositivos móviles, tablets y escritorio.

Importante

Dentro del documento PDF se encuentra detallada la consigna completa, incluyendo los enlaces a las imágenes que deberán utilizar para el desarrollo del sitio.
